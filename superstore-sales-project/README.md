# 📊 Superstore Sales Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing retail sales data from a superstore to uncover key business insights. The goal is to transform raw data into a clean and interactive dashboard that helps understand sales performance, customer behavior, and product trends.

---

## 💼 Business Problem
How can the company improve sales performance and identify the most profitable categories and customer segments based on historical data?

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing product categories
- Understand customer segment contribution
- Track sales trends over time

---

## 📂 Dataset Information
- Dataset: Superstore Sales Dataset (Kaggle)
- Records: 9,794 rows
- Features:
  - Order Date
  - Sales
  - Category
  - Segment
  - Order ID

---

## 🛠 Tools & Techniques
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Data Visualization
- Dashboard Design

---

## 🔍 Data Analysis Process

### 1. Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized number formats (sales column)

### 2. Data Transformation
- Created calculated fields (AOV, Total Orders)
- Extracted Year and Month from Order Date
- Aggregated data using Pivot Tables

### 3. Data Visualization
- Line chart for trend analysis
- Bar chart for category comparison
- Donut chart for segment distribution

---

## 💡 Key Insights
- Sales show a consistent upward trend from 2015 to 2018, although growth is gradual
- Technology category generates the highest revenue compared to other categories
- Consumer segment contributes more than 50% of total sales
- Seasonal spikes appear in certain months, indicating potential promotional periods

---

## 🚀 Business Recommendations
- Increase focus on Technology products as they generate the highest revenue
- Target Consumer segment with more personalized promotions
- Leverage seasonal trends to optimize marketing campaigns and boost sales

---

## 📌 Conclusion
This project demonstrates how Excel can be used not only for data processing but also for building interactive dashboards that support data-driven business decisions.
